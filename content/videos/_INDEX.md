# Index des Vidéos YouTube

Transcriptions de vidéos YouTube organisées avec tags.

## Organisation

Chaque vidéo est sauvegardée avec :
- **Titre** : Titre descriptif
- **video_id** : ID YouTube
- **date** : Date d'ajout
- **url** : Lien YouTube
- **tags** : Catégories (dev, business, tech, music, etc.)

## Tags courants

- `dev` - Développement / Programmation
- `python` - Python
- `ai` - Intelligence Artificielle
- `business` - Business / Entrepreneuriat
- `tech` - Technologie générale
- `music` - Musique / Audio
- `tutorial` - Tutoriels
- `conference` - Conférences

## Utilisation

```bash
# Ajouter une vidéo
yt VIDEO_ID --save --title "Titre" --tags dev,python
```
